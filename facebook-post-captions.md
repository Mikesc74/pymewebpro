## Posting strategy (read first)

- **Spam protection:** Don't post identical text to multiple groups the same day. FB flags duplicate cross-group posts. Swap the hook line or rephrase a sentence for each group.
- **Link rules:** Many groups ban links in post body. If rejected, put the website URL in the first comment; keep WhatsApp in the post (it's often allowed since it's not "external").
- **Hashtags:** FB hashtags are mostly cosmetic — 3–5 max. Don't overdo it.
- **Best posting times in Colombia:** Tue/Thu around 11am–1pm or 7–9pm tend to perform best for local SMB audiences.

## Quick variation tricks (for posting same flyer to multiple groups)

To dodge FB's spam detector, swap one of these per group:

- **Hook rephrasing:** "¿Cuántos clientes pierde…" → "¿Sabe cuántos clientes pierde…"
- **CTA:** "hablemos" → "escríbanos por WhatsApp" → "le ayudamos hoy mismo"
- **Open with location:** "Para PyMEs en Bogotá:" / "Para negocios en Cali:" / "Para emprendedores en Medellín:"
- **Reorder paragraphs:** lead with price, or lead with the guarantee
- **Emoji swap:** add or remove a 🚀 / ✨ / 💼 / ✅
